# Barcode_Creatore_Reader
BarCode e QRCode creator and reader. 
# What to install 
It depends on the file, anyway for all the codes it's necesary to install: 
* python-barcode
* qrcode
* cv2
* pyzbar 
* imutils
* argparse
* datetime
* pygame
